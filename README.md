## 내부 조인
내부 조인은 둘 이상의 테이블에서 공통된 값을 기준으로 데이터를 결합하는 방법 중 하나이다. <br><br>

## 셀프 조인
셀프 조인은  1개의 테이블(X)에 가상으로 x1, x2라는 별칭을 부여하여 2개의 테이블인 것처럼 간주한 뒤 JOIN하는 것이다.  <br><br>
